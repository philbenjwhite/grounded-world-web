# Welcome to our TinaCMS Blog

This is a sample blog post created with TinaCMS. You can edit this content directly through the TinaCMS admin interface.

## Features

- **Visual Editing**: Edit your content in real-time
- **Git-Based**: All content is stored in your repository
- **Markdown Support**: Write content in Markdown
- **Tailwind CSS**: Beautiful styling with Tailwind

## Getting Started

1. Run `npm run dev` to start the development server
2. Visit `/admin` to access the TinaCMS admin panel
3. Start editing your content!

Happy writing!
