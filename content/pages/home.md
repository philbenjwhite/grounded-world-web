# Welcome to Grounded World

This is a Next.js application powered by TinaCMS and styled with Tailwind CSS.

## About This Site

This website is built with modern web technologies:

- **Next.js** - React framework for production
- **TinaCMS** - Git-backed headless CMS
- **Tailwind CSS** - Utility-first CSS framework

Edit this content through the TinaCMS admin panel at `/admin`.
